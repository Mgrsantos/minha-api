# 🚀 Minha API REST - Spring Boot

API RESTful desenvolvida com Spring Boot, com autenticação JWT e persistência em PostgreSQL.

## 🛠️ Tecnologias
- Java 17
- Spring Boot 3
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Swagger (OpenAPI 3)
- Bean Validation

## 📋 Funcionalidades
- CRUD completo de Produtos
- Cadastro e autenticação de usuários
- Proteção de endpoints com JWT
- Validação de dados
- Tratamento global de erros
- Documentação automática via Swagger
